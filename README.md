## 如何使用
Clone此仓库，然后将Core部分的文件夹覆盖复制到您的PE项目文件夹中，然后进行问题检修工作
>我们不建议缺乏Windows系统和WinPE基础知识的使用者进行此工作

此外，您的PE需要添加PECMD支持并完成构建，我们推荐使用Wimbuilder系列软件构建您的PE核心
